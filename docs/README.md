# TASHA

**Tasha** is a tool for creating player characters for the 5.5e *Dungeons & Dragons (2024)* tabletop game. It is a rewrite of a program that I originally wrote in python.

## Usage

```
go build
```

Get the current working version using the following command.

```
./tasha version
```

Create a new character using the following command.

```
./tasha new <CHARACTER_NAME>
```
